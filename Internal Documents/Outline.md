# Features

## Gameplay
- Multiple Playstyles
    - Solo
    - Multiplayer
        - Coop
        - Competitive
- Multiple Game Modes
    - Traditional Bingo
    - Blackout
    - Lockout
    - Rush
- Game Settings
    - Default Presets
    - Saveable Preset System
    - Adjustable Luck Reliance
    - Difficulty Options
- Dynamic Objective System
    - Custom Objective Support
    - Objective Category System
        - Weight Categories
        - Baked in Luck ratings
        - Per-Objective Difficulty Scoring

## UI
- 4 Board Sizes
    - Full Screen
    - Large
    - Small
    - Hidden (tab)
- Resize/Reposition Boards (Small/Large/Tab Placement(Hidden))
    - Setting to decouple Large/Small Board positions (sync/desycn board locations)

## Planned
- Player Username (persistent)
- Player Color (set it settings for solo/set in lobby for multiplayer (once mod detects lobby))

## Future Scope
- Auto-Detecting Objective Completion
- Objective Completion Rewards
- Ascend Game Mode
- Auto-Detection with Custom Objective Lists
